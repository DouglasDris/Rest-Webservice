# Simple Java Server

## Products API

[X] /GET /api/v1/Products
 - Returns all products
 
[X] /GET /api/v1/Products/:id
 - Return product by ID

[X] /PUT /api/v1/Products/:id
 - Update product by ID
 
[X] /POST /api/v1/Products
 - Create Product
 
[X] /DELETE /api/v1/Products/:id
 - Delete product by ID
